 Análisis Predictivo de Ventas - Modelos a Escala
**Proyecto de Ingeniería de Sistemas - 3er Semestre

Este repositorio contiene un pipeline completo de Ciencia de Datos aplicado a un dataset de ventas internacionales.

  Tecnologías y Herramientas
  Lenguaje: Python 3.x
  Entorno: Google Colab
  Librerías: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn.

Fases del Proyecto
1. Limpieza (ETL): Eliminación de registros nulos y duplicados.
2. **Preprocesamiento: Escalación de datos con `MinMaxScaler` para normalizar las variables de entrada.
3. **Análisis Estadístico: Estudio de tendencias centrales y detección de outliers mediante Boxplots y Violín plots.
4. **Modelado:Implementación de un algoritmo de Regresión Logística con una precisión del 90%.

#Cómo ejecutar
1. Clonar el repositorio.
2. Subir el archivo `.csv` a la sesión de Colab.
3. Ejecutar las celdas en orden secuencial.
